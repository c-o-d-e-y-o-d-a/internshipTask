
# Delv

Assignments I need to complete for internship selection processes



## Run Locally

Clone the project

```bash
  git clone https://github.com/c-o-d-e-y-o-d-a/flutter-delivery-app
```

Go to the project directory

```bash
  cd [ desired_folder]
```

Install dependencies

```bash
  flutter pub get
```

Start the app

```bash
  flutter run 
```
